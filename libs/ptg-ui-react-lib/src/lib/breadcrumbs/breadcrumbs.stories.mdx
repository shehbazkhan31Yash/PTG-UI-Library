# PtgUiBreadcrumbs

A functional component that renders a breadcrumb navigation list.

## Props

| Name         | Type     | Description                                      |
|--------------|----------|--------------------------------------------------|
| `datalist`   | `Array`  | An array of breadcrumb items.                    |
| `handleClick`| `Function` | A function to handle click events on breadcrumb links. |

## Example

<Canvas>
  <Story name="Default">
    <Router>
      <PtgUiBreadcrumbs
        datalist={[
          { title: 'Home', link: '/' },
          { title: 'Library', link: '/library' },
          { title: 'Data', link: '/data' },
        ]}
        handleClick={() => alert('Breadcrumb clicked!')}
      />
    </Router>
  </Story>
</Canvas>

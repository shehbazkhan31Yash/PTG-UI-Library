# Loading Component

---

The Loading component provides different styles of animated loaders to indicate ongoing processes.

## Properties

### Loader Controls:
- **type** – Defines the loader type (`dot`, `linear`, `circular`).
- **color** – Sets the loader's color.

## Usage Example

```tsx
import { PtgUiLoading } from '@ptg-ui/react';

<PtgUiLoading 
  type="dot"
  color="#3498db"
/>

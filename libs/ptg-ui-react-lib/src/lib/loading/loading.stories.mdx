# PtgUiLoading

The `PtgUiLoading` component is a customizable loader that can display different types of loading indicators. It supports three types of loaders: **dot**, **linear**, and **circular**.

## Usage

To use the `PtgUiLoading` component, import it and provide the necessary props.

```jsx
import { PtgUiLoading } from './PtgUiLoading';

<PtgUiLoading type="dot" color="#3498db" />

# PtgUiAppBar Component

The `PtgUiAppBar` component renders a customizable navigation bar for applications. It offers flexible menu configurations, supports both dropdown and drawer-style burger menus, and provides options to align the logo and menu elements.

---

## **Usage**
```tsx
import { PtgUiAppBar } from './AppBar';

<PtgUiAppBar
  menuConfig={{
    logo: 'https://your-logo-url.com/logo.png',
    menuItems: <ul><li>Home</li><li>About</li><li>Contact</li></ul>,
    burgerMenu: true,
    burgerMenuType: 'dropdown',
    position: 'top',
  }}
  openMenu={false}
/>;

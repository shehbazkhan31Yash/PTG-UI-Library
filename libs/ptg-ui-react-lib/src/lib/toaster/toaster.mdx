# Toaster Component

---

The Toaster component displays notifications to users with different message types such as success, error, warning, and info.

## Properties

### Toaster Controls:
- **message** – Defines the notification message content.
- **type** – Specifies the type of toaster (`success`, `error`, `info`, `warning`).
- **showDescription** – Boolean to enable or disable the description in the toaster.
- **alignItem** – Controls the alignment (`flex-start`, `center`, `flex-end`).
- **justifyContent** – Determines how the toaster content is justified (`flex-start`, `center`, `flex-end`).

## Usage Example

```tsx
import { PtgUiToaster } from '@ptg-ui/react';

<PtgUiToaster 
  message="This is a sample toaster message."
  type="success"
  showDescription={true}
  alignItem="center"
  justifyContent="center"
/>

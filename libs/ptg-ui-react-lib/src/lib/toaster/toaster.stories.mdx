# PtgUiToaster Component

The `PtgUiToaster` component is a customizable notification popup for your applications. It supports various types of messages and styles.

You can import the component using the following statement:
import { PtgUiToaster } from './toaster';

# message
The message prop sets the text content displayed in the toaster.

Example: <Canvas> <Story id="components-toaster--primary" /> </Canvas>

#type
The type prop determines the style of the toaster. Possible options include error, success, info, and warning.

Example: <Canvas> <Story id="components-toaster--primary" /> </Canvas>

# showDescription
The showDescription prop determines if an additional description or message is shown in the toaster.

Example: <Canvas> <Story id="components-toaster--primary" /> </Canvas>

# alignItem
The alignItem prop controls the vertical alignment of the toaster.

Example: <Canvas> <Story id="components-toaster--primary" /> </Canvas>

# justifyContent
The justifyContent prop controls the horizontal alignment of the toaster.

Example: <Canvas> <Story id="components-toaster--primary" /> </Canvas>

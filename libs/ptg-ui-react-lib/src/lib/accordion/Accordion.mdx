# Accordion Component

---

A customizable accordion component for collapsible sections.

## Properties

### Accordion Controls:
- **accordionItems** – Array of objects with title and content for each section.
- **handleToggle** – Callback function triggered when an accordion item is clicked.
- **activeIndex** – Index of the currently active accordion item.

## Usage Example

```tsx
import { PtgUiAccordion } from '@ptg-ui/react';

<PtgUiAccordion 
  accordionItems={[
    { title: 'Section 1', content: 'This is the content of section 1.' },
    { title: 'Section 2', content: 'This is the content of section 2.' },
    { title: 'Section 3', content: 'This is the content of section 3.' },
  ]}
  activeIndex={0}
  handleToggle={(index) => console.log('Toggled section:', index)}
/>

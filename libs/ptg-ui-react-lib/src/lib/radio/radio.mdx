# Radio Component

---

The Radio component provides a set of options where only one selection is allowed at a time.

## Properties

### Radio Controls:
- **name** – The name attribute for the radio input group.
- **value** – The selected radio input value.
- **id** – The ID prefix for radio inputs.
- **list** – Array of radio options including `id`, `name`, and `value`.

## Usage Example

```tsx
import { PtgUiRadio } from '@ptg-ui/react';

<PtgUiRadio 
  name="radio-group"
  id="radio-id"
  list={[
    { id: '1', name: 'Option 1', value: '1' },
    { id: '2', name: 'Option 2', value: '2' },
    { id: '3', name: 'Option 3', value: '3' },
  ]}
/>

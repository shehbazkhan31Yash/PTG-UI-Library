# PtgUiInput Component

The `PtgUiInput` component is a versatile and customizable input field designed for various use cases such as text entry, placeholders, and more.

You can import the component using the following statement:
```jsx
import PtgUiInput from './input';

The PtgUiInput component supports the following controls:

# type
The type prop specifies the input type, such as text, password, or email. It determines the kind of data that can be entered.

Example:

# value
The value prop represents the current value of the input field. It can be controlled programmatically or used for initial values.

Example:
<Canvas> <Story id="components-inputcomponent--default" /> </Canvas>

# placeholder
The placeholder prop displays a hint or instructional text inside the input field when no value is entered.

Example:
<Canvas> <Story id="components-inputcomponent--with-placeholder" /> </Canvas>

# disabled
The disabled prop makes the input field uneditable and visually indicates its state.

Example:
<Canvas> <Story id="components-inputcomponent--disabled" /> </Canvas>

# required
The required prop marks the input field as mandatory. Form submission will not proceed unless a value is entered.

Example:
<Canvas> <Story id="components-inputcomponent--required" /> </Canvas>

# maxlength
The maxlength prop limits the maximum number of characters that can be entered in the input field.

Example:
<Canvas> <Story id="components-inputcomponent--max-length" /> </Canvas>

# className
The className prop allows custom styles to be applied to the input element, enhancing its appearance.

Example:
<Canvas> <Story id="components-inputcomponent--default" /> </Canvas>

# inputsize
The inputsize prop adjusts the size of the input field. Options include sm, md, and lg.

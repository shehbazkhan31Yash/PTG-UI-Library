# Input Component

---

The Input component allows users to enter text with various properties such as size, placeholders, and validation controls.

## Properties

### Input Controls:
- **type** – Defines the input type (e.g., text, email, password).
- **value** – Specifies the input field's current value.
- **placeholder** – Placeholder text displayed within the input field.
- **disabled** – Boolean to disable input interaction.
- **required** – Boolean indicating if input is required.
- **className** – Custom styling class for the input field.
- **inputsize** – Defines the size of the input field (`sm`, `md`, `lg`).
- **name** – Name attribute for the input field.
- **maxlength** – Maximum length of input characters.

## Usage Example

```tsx
import { PtgUiInput } from '@ptg-ui/react';

<PtgUiInput 
  type="text"
  value=""
  placeholder="Enter your name"
  disabled={false}
  required={true}
  className="input-style"
  inputsize="md"
  name="nameInput"
  maxlength="20"
/>

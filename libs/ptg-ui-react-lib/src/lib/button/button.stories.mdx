# PtgUiButton Component

---

The `PtgUiButton` is a customizable and reusable button component designed for various use cases, offering size, style, and alignment customizations.

You can import the component using the following statement:
```jsx
import { PtgUiButton } from '@ptg-ui/react';

## Quick Start

Here’s an example of how you can create a button in your application:

```jsx
<PtgUiButton 
  appearance="primary" 
  text="Click Me" 
  width="150px" 
  height="40px" 
  fontSize="16px" 
  fontWeight="500" 
  textColor="#ffffff" 
  backgroundColor="#007bff" 
/>

## Types of Controls Supported by PtgUiButton

1.] Appearance

The `appearance` prop defines the style of the button (e.g., primary, secondary, warning, success, danger, info, light, dark). This allows you to create buttons that visually align with your application's theme.

3.] Disabled State
The disabled prop allows you to visually render a button in an inactive state. Disabled buttons are styled with a lighter appearance to differentiate them from active buttons.

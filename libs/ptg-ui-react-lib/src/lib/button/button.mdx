# Button Component

---

A customizable button component with different appearances, sizes, and interaction options.

## Properties

### Button Controls:
- **appearance** – Defines the button style (`primary`, `secondary`, `warning`, `success`, `danger`, `info`, `light`, `dark`).
- **text** – Sets the button's displayed text.
- **btnIconAlignment** – Specifies icon alignment (`left`, `right`).
- **hasbtnIconSlot** – Boolean indicating whether the button includes an icon slot.
- **width** – Sets the width of the button.
- **height** – Defines the height of the button.
- **fontSize** – Specifies the font size of the button text.
- **fontWeight** – Controls the font weight of the button text.
- **textColor** – Defines the button text color.
- **backgroundColor** – Sets the background color of the button.
- **disabled** – Boolean indicating whether the button is disabled.
- **type** – Defines the button type (e.g., `button`, `submit`).
- **border** – Specifies the button border style.
- **onClick** – Event triggered when the button is clicked.

## Usage Example

```tsx
import { PtgUiButton } from '@ptg-ui/react';

<PtgUiButton 
  appearance="primary"
  text="Click Me"
  width="200px"
  height="60px"
  fontSize="20px"
  fontWeight="600"
  textColor="#ffffff"
  backgroundColor="#2196f3"
  disabled={false}
/>

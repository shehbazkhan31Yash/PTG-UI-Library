# PtgUiPagination Component

The `PtgUiPagination` component is used to navigate through pages of data efficiently. It supports customizable page sizes, sibling count, and navigation controls.

## **Usage**
```tsx
import { PtgUiPagination } from './pagination';

<PtgUiPagination
  dataCount={100}
  pageNumber={1}
  pageSize={10}
  siblingCount={1}
  previousBtnText="Previous"
  nextBtnText="Next"
  pageIndex={(page) => console.log(`Page changed to ${page}`)}
/>;

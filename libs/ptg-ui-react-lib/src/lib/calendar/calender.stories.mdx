# PtgUiCalendar Component

The `PtgUiCalendar` component is a versatile date and datetime picker designed to integrate seamlessly into your React applications. With options for customization and accessibility, it empowers users to select dates efficiently.

You can import the component using the following statement:
```jsx
import { PtgUiCalendar } from '@ptg-ui/react';


## Features

The `PtgUiCalendar` component supports the following controls:

### `className`

The `className` attribute allows you to apply custom styles to the calendar input field.

The `startDate` prop defines the minimum selectable date or datetime. Use this to restrict past dates or set specific boundaries.

# Calendar Component

---

The Calendar component allows users to select and display dates with optional datetime functionality.

## Properties

### Calendar Controls:
- **className** – Custom class name for styling the calendar input.
- **selected** – Defines the currently selected date or datetime.
- **disabled** – Boolean that determines if the calendar input should be disabled.
- **isDateTime** – Boolean that indicates whether to enable datetime selection.

## Usage Example

```tsx
import { PtgUiCalendar } from '@ptg-ui/react';

<PtgUiCalendar 
  className="default-calendar" 
  selected={new Date()} 
  disabled={false} 
  isDateTime={false}
/>

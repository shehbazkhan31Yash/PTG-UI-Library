# PtgUiCarousel Component

The `PtgUiCarousel` component allows you to create a flexible and interactive carousel with customizable navigation options. You can use it for images, cards, or any custom items.

---

## **Usage**

```tsx
import { PtgUiCarousel } from './carousel';

<PtgUiCarousel
  items={[
    { id: '1', image: 'https://via.placeholder.com/300', title: 'Slide 1', description: 'Description 1' },
    { id: '2', image: 'https://via.placeholder.com/300', title: 'Slide 2', description: 'Description 2' },
  ]}
  backgroundColor="#fff"
  buttonPosition="bottom"
  buttonProps={{
    iconLeft: '<',
    iconRight: '>',
    style: { backgroundColor: '#007bff', color: '#fff', width: '40px', height: '40px' },
  }}
  navigationOnIcon={false}
  navigationIconWidth="40px"
  navigationIconHeight="40px"
/>;

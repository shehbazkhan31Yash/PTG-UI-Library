# PtgUiTextArea Component

The `PtgUiTextArea` component is a customizable textarea that allows users to input multi-line text.

You can import the component using the following statement:
import { PtgUiTextArea } from './textarea';

The PtgUiTextArea component supports the following controls:

# name
The name prop specifies the name attribute for the textarea, which is useful for form data handling.

Example: <Canvas> <Story id="components-textarea--default" /> </Canvas>

# value
The value prop specifies the current value of the textarea, which can be controlled programmatically.

Example: <Canvas> <Story id="components-textarea--value" /> </Canvas>

# id
The id prop assigns a unique identifier to the textarea for accessibility and interaction purposes.

Example: <Canvas> <Story id="components-textarea--id" /> </Canvas>

# className
The className prop allows you to apply custom styles to the textarea for seamless integration with your design.

Example: <Canvas> <Story id="components-textarea--class-name" /> </Canvas>

# placeholder
The placeholder prop displays instructional or hint text when the textarea is empty.

Example: <Canvas> <Story id="components-textarea--placeholder" /> </Canvas>

# rows
The rows prop specifies the number of visible text lines in the textarea.

Example: <Canvas> <Story id="components-textarea--rows" /> </Canvas>

# onChange
The onChange prop is a callback function triggered when the content of the textarea changes.

Example: <Canvas> <Story id="components-textarea--on-change" /> </Canvas>

# onBlur
The onBlur prop is a callback function triggered when the textarea loses focus.

Example: <Canvas> <Story id="components-textarea--on-blur" /> </Canvas>

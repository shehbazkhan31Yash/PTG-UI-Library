# TextArea Component

---

The TextArea component provides a multi-line text input field for users to enter longer content.

## Properties

### TextArea Controls:
- **placeholder** – Defines the placeholder text displayed inside the textarea.
- **rows** – Specifies the number of visible rows in the textarea.
- **className** – Custom styling class for the textarea.
- **name** – Defines the name attribute of the textarea.
- **id** – Unique identifier for the textarea.
- **value** – Represents the pre-filled value of the textarea.

## Usage Example

```tsx
import { PtgUiTextArea } from '@ptg-ui/react';

<PtgUiTextArea 
  placeholder="Enter your message..."
  rows={4}
  className="custom-textarea"
  name="message"
  id="messageId"
  value=""
/>

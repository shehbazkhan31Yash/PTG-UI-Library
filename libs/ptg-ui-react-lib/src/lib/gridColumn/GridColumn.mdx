# PtgUiGridColumn Component

The `PtgUiGridColumn` component allows you to organize content in a responsive grid layout. It supports various screen sizes with the ability to span columns and add offsets.

---

## **Usage**

### Basic Example

```tsx
import { PtgUiGridColumn } from './GridColumns';
import { PtgUiRow } from '../gridRow/GridRow';
import { PtgUiCard } from '../card/Cards';

<PtgUiRow>
  <PtgUiGridColumn xl={4} lg={6} md={6} sm={12} xs={12}>
    <PtgUiCard
      image="https://via.placeholder.com/150"
      title="Dynamic Card Title"
      description="Dynamic Card Description"
      buttonText="Learn More"
      buttonColor="#00f"
      buttonWidth="150px"
    />
  </PtgUiGridColumn>
</PtgUiRow>;

# Rating Component

---

The Rating component allows users to provide feedback using a star-based system or custom icons.

## Properties

### Rating Controls:
- **value** – Defines the rating value.
- **readOnly** – Boolean to make the rating read-only.
- **disabled** – Boolean to disable interaction.
- **precision** – Defines the step value for rating increments.
- **icon** – Custom icon representing a filled rating.
- **emptyIcon** – Custom icon for an empty rating state.
- **color** – Sets the color of the rating icons.
- **borderColor** – Defines the border color for rating icons.
- **size** – Sets the size of the icons.
- **hoverSize** – Defines the size when hovering over icons.
- **defaultValue** – Specifies the initial rating value.
- **onChange** – Action triggered when rating value changes.
- **onHover** – Action triggered when hovering over rating options.

## Usage Example

```tsx
import { Rating } from '@ptg-ui/react';

<Rating 
  value={3}
  readOnly={false}
  disabled={false}
  precision={1}
  icon="★"
  emptyIcon="☆"
  color="#FFD700"
  borderColor="#ccc"
  size={24}
  hoverSize={28}
/>

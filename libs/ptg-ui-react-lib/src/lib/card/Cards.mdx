# Card Component

---

The Card component is a customizable UI element used to display content such as images, text, and interactive buttons.

## Properties

### Card Controls:
- **onClick** – Action triggered when the button is clicked.
- **backgroundColor** – Defines the background color of the card.
- **buttonColor** – Sets the button color.
- **buttonTextColor** – Determines the text color of the button.

## Usage Example

```tsx
import { PtgUiCard } from '@ptg-ui/react';

<PtgUiCard 
  image="https://picsum.photos/id/1/200/300"
  title="Example Card"
  description="This is a customizable card component."
  buttonText="Click Here"
  buttonColor="#007bff"
  buttonTextColor="#ffffff"
  backgroundColor="#f8f9fa"
/>

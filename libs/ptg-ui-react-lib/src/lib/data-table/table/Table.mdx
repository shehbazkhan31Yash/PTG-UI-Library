# PtgUiTable Component

The `PtgUiTable` component is a powerful and customizable table component that supports features like filtering, sorting, pagination, sticky headers, and server-side data handling.

---

## **Usage**

### Basic Table Example
```tsx
import { PtgUiTable } from './Table';

<PtgUiTable
  columns={[
    { accessor: 'name', Header: 'Name' },
    { accessor: 'age', Header: 'Age' },
  ]}
  data={[
    { id: 1, name: 'John Doe', age: 28 },
    { id: 2, name: 'Jane Smith', age: 34 },
  ]}
/>;

# Checkbox Component

---

The Checkbox component allows users to toggle between checked and unchecked states. It supports labels and customizable properties.

## Properties

### Checkbox Controls:
- **id** – Unique identifier for the checkbox.
- **name** – Name attribute for the checkbox.
- **value** – Value assigned to the checkbox input.
- **className** – Custom class name for styling.
- **htmlFor** – Specifies the `for` attribute for labeling purposes.
- **checked** – Boolean defining the default checked state.

## Usage Example

```tsx
import { PtgUiCheckbox } from '@ptg-ui/react';

<PtgUiCheckbox 
  id="custom-checkbox" 
  name="customCheckbox"
  value="custom"
  className="checkbox-style"
  htmlFor="custom-checkbox"
  checked={false}
/>

# PtgUiCheckbox Component

The `PtgUiCheckbox` component is a flexible and user-friendly input designed for toggling options.

You can import the component using the following statement:
```jsx
import { PtgUiCheckbox } from './checks';

The PtgUiCheckbox component supports the following controls:

# checked
The checked prop specifies whether the checkbox is checked or not. It can be controlled programmatically or used for initial values.

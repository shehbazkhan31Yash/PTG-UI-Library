# Alert Component

---

A reusable alert component for displaying different types of messages.

## Properties

### Alert Controls:
- **type** – Defines the alert type (`success`, `warning`, `danger`, `info`).
- **message** – Specifies the message displayed inside the alert.

## Usage Example

```tsx
import { PtgUiAlert } from '@ptg-ui/react';

<PtgUiAlert 
  type="success"
  message="Operation completed successfully!"
/>

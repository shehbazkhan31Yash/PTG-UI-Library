# PtgUiDatePicker

The `PtgUiDatePicker` component is a custom date picker that allows users to select a date from a calendar popup.

## Usage

To use the `PtgUiDatePicker`, import it and provide the necessary props.

```jsx
import { PtgUiDatePicker } from './date-picker';

const handleDateSelect = (date) => {
  console.log('Selected date:', date);
};

<PtgUiDatePicker sendSelectedDate={handleDateSelect} id="date-picker" className="custom-class" />

# Datepicker Component

---

The Datepicker component allows users to select and display dates efficiently.

## Properties

### Datepicker Controls:
- **id** – Unique identifier for the datepicker.
- **className** – Custom class name for styling.
- **sendSelectedDate** – Callback function triggered when a date is selected.

## Usage Example

```tsx
import { PtgUiDatePicker } from '@ptg-ui/react';

<PtgUiDatePicker 
  id="custom-date-picker" 
  className="datepicker-style"
  sendSelectedDate={(date) => console.log('Selected date:', date)}
/>

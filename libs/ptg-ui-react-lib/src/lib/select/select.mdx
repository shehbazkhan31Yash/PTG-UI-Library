# Select Component

---

The Select component provides a dropdown menu for users to choose from predefined options.

## Properties

### Select Controls:
- **name** – Defines the name attribute for the select input.
- **value** – Represents the selected value.
- **id** – Unique identifier for the select input.
- **className** – Custom styling class for the dropdown.
- **list** – Array of selectable options.
- **placeholder** – Displays a placeholder text in the dropdown.
- **disabled** – Boolean to disable the dropdown interaction.

## Usage Example

```tsx
import { PtgUiSelect } from '@ptg-ui/react';

<PtgUiSelect 
  name="example-select"
  id="select-id"
  className="example-select"
  list={[
    { id: '1', label: 'Option 1', value: '1' },
    { id: '2', label: 'Option 2', value: '2' },
    { id: '3', label: 'Option 3', value: '3' },
  ]}
  placeholder="Select an option"
  disabled={false}
/>

# Modal Component

---

The Modal component provides a customizable pop-up dialog with optional headers, footers, and action buttons.

## Properties

### Modal Controls:
- **isOpen** – Determines whether the modal is visible.
- **modalSize** – Defines the modal size (`sm`, `md`, `lg`).
- **showHeader** – Boolean to show or hide the modal header.
- **showFooter** – Boolean to show or hide the modal footer.
- **confirmButton** – Text for the confirm button.
- **cancelButton** – Text for the cancel button.
- **backdropClick** – Allows closing the modal by clicking the backdrop.
- **content** – Specifies the main content inside the modal.
- **confirmButtonColor** – Defines the confirm button color.
- **cancelButtonColor** – Defines the cancel button color.

## Usage Example

```tsx
import { PtgUiModal } from '@ptg-ui/react';

<PtgUiModal 
  isOpen={true}
  modalSize="md"
  showHeader={true}
  header="Example Modal"
  showFooter={true}
  confirmButton="Confirm"
  cancelButton="Cancel"
  backdropClick={true}
  content="This is an example of a modal component."
  confirmButtonColor="#2196f3"
  cancelButtonColor="#dd3434"
/>

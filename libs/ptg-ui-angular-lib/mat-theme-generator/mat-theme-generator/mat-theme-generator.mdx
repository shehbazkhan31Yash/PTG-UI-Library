# MatThemeGenerator Component

This component allows users to generate and save theme colors.

We can import the Theme tag using the following importation :

```
    import { MatThemeGeneratorModule } from '@ptg-ui/angular/mat-theme-generator'
```
## Quick Start

we can create a theme-generator using the following HTML Tag:

```
  <ptg-ui-mat-theme-generator></ptg-ui-mat-theme-generator>`;


```

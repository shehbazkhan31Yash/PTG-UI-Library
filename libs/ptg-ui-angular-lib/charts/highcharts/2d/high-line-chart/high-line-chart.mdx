# HighLineChart Component

---

This component renders a 2D line chart using the [Highcharts](https://www.highcharts.com/) library.

A line chart is commonly used to show trends over time or categories using connected points.

---

## 📦 Import

```ts
import { HighLineChartComponent } from '@ptg-ui/angular';


```

Supported Inputs :

```
| Input             | Type       | Description                                                         |
| ----------------- | ---------- | ------------------------------------------------------------------- |
| `data`            | `Array`    | Array of data series to be plotted on the chart. Required.          |
| `categories`      | `string[]` | X-axis categories (labels for each point on the X-axis). Optional.  |
| `isCreditEnabled` | `boolean`  | Whether to display the Highcharts credit label. Default is `false`. |

```
## Quick Start
 
 we can create a highLineChart using the following tag: 

```
html

<ptg-ui-high-line-chart
  [data]="data"
></ptg-ui-high-line-chart>


Where lineChartData is an array like:
ts
[
  {
    name: 'Year 1800',
    data: [7.0, 6.9, 9.5]
  }
]

```

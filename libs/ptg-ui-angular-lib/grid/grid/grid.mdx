# Grid Component

---

Grid system uses a series of containers, rows, and columns to layout and align content. It’s built with flexbox and is fully responsive. Below is an example and an in-depth explanation for how the grid system comes together.


We can import the grid layout tag using the following importation :

```
    import { GridModule } from '@ptg-ui/angular/grid'
```

Types of controls supported by card:

- columnClasses
- columnClass
- rowClass

## Quick Start

we can create a grid layout using the following HTML Tag:

```
<ptg-ui-grid [columnClasses]="gridOffsetExampleColumnClass">
            <ng-template>Your Column Template</ng-template></ptg-ui-grid>

```

Following are some controls with their explanation :

1] columnClasses , columnClass ,rowClass:

 A]columnClasses : grid system can adapt across all six default breakpoints, and any breakpoints you customize. The six default grid tiers are as follow:

Extra small (xs)
Small (sm)
Medium (md)
Large (lg)
Extra large (xl)

User need to send count for all these breakpoints also can provide offset if require. 
Number of template will be same as column count.
B] columnClass :  User can send any custom class for column of grid.

B] rowClass :  User can send any custom class for row of grid.

# 📌 Sidenav Component

The `SidenavComponent` provides a responsive and collapsible side navigation menu. It adapts based on the screen size and allows navigation via menu items, including nested submenus.

---

## ✅ Import

```ts
import { SidenavComponent } from '@ptg-ui/angular';

```

⚙️ Component Inputs
```
| Input          | Type      | Default | Description                                  |
| -------------- | --------- | ------- | -------------------------------------------- |
| `isMenuOpen`   | boolean   | false   | Determines if the sidenav is expanded or not |
| `menuItems`    | object\[] | `[]`    | List of menu items to render                 |
| `selectedPath` | string    | `''`    | Currently active/selected route path         |

```
🔁 Component Outputs
```

| Output       | Description                       |
| ------------ | --------------------------------- |
| `selectMenu` | Emits selected menu item on click |
```

💡 Usage
```
<ptg-ui-sidenav1 
  [isMenuOpen]="true" 
  [menuItems]="menuList" 
  (selectMenu)="onMenuSelect($event)">
</ptg-ui-sidenav1>
```
